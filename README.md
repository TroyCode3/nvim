# Neovim config file

## Get started:
Run `/setup/install.sh` (for Arch-based only)

## Install Mason packages:
* jsonlint
* shellcheck
* stylua
* prettier

## Language-based packages:
* Markdown:
    + Compiler: pandoc
* Latex:
    + Compiler: texlive-most
    + LSP: texlab
* C/C++:
    + Make utility: bear
* Rust:
    + LSP: rust_analyzer
